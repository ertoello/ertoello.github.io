About


Hallo, saya erto. 3 tahun terakhir saya sedang menempuh studi di jurusan teknologi informasi. saya akan menceritakan tentang apa yang saya sudah dapatkan selama 3 tahun selama perkulihaan di kampus saya. Selama 3 tahun terakhir ini, saya memahami 4 bahasa pemrogaman, yaitu; python, java, PHP, dan JavaScript. Bahasa pertama yang saya pernah belajar adalah Python. saya belajar bahasa Python waktu semester 1, dilanjutkan dengan bahasa pemrogaman java pada semester 2, dan dlanjutkan bahasa pemrogaman PHP dan JavaScript. Dari keempat bahasa pemrogaman ini, sepertinya saya lebih tertarik bahasa pemrogaman PHP dan JavaScript. Karna bahasa pemrogaman tersebut adalah bahasa untuk membuat website-website. Karna saya lebih tertarik membuat website. Oleh karena itu, saya ingin memperdalam materi bahasa pemrogaman sebagai bekal utk magang, skripsi, dan terjun ke dunia kerja nantinya.

Selama masa perkuliahan, akhirnya saya menemukan minat saya dimana saya terus mencari apa yang menjadi cocok pemrogaman dan harus diperdalam programnya.
